# Container Queries

:::caution

Container Query support is experimental! We are waiting for your feedback to improve implementation.

:::

NativeWind supports the official [TailwindCSS Container Query plugin](https://github.com/tailwindlabs/tailwindcss-container-queries)

<Compatibility
  supported={[
    "@container",
    "@container/{name}",
    "@{size}/{name}:{modifier}",
    "@container-normal",
  ]}
/>
